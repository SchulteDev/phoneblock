Android installation

The installation adds the PhoneBlock address book to your contacts. If you receive a call from a spam number, this will be displayed immediately, e.g. "SPAM: 03016637169". You can also see immediately in the missed calls that it is not worth calling back without first googling the number.

You cannot use this to block SPAM calls (your phone will still ring, but you will see that it is a SPAM call). You can block SPAM calls by installing SpamBlocker and integrating the PhoneBlock database. SpamBlocker is an independent app, please contact the developer directly if you have any problems with it.

For address book synchronization, you need an app with which you can synchronize Internet phone books. For example, you can use "PeopleSync" or "DAVx⁵". The instructions here are written for "PeopleSync", but the procedure with "DAVx⁵" is very similar:

Step 1: Open the Playstore and search for "PeopleSync"
