Installation de l'iPhone

L'installation ajoute le carnet d'adresses PhoneBlock à tes contacts. Si tu reçois un appel d'un numéro de spam, tu en seras immédiatement informé, par exemple "SPAM : 03016637169". De même, tu vois tout de suite dans les appels manqués qu'il ne vaut pas la peine de rappeler ici sans googler d'abord le numéro.

L'installation se déroule selon les étapes suivantes :

  1. Inscris-toi sur PhoneBlock.
  2. Ouvre "Paramètres" > "Contacts" > "Comptes".
  3. Tape sur "Ajouter un compte" - "Autres" > "Ajouter un compte CardDAV".
  4. Saisis tes données d'accès et tape sur "Suivant".
    Serveur
    https://phoneblock.net/contacts/
    Nom d'utilisateur
    Connecte-toi pour voir ton nom d'utilisateur.
    Mot de passe
    Il t'a été montré directement après ta première connexion, tu peux en générer un nouveau dans les paramètres.
    Description
    PhoneBlock
  5. Active "Contacts" et tape sur "Sauvegarder".
  6. Tu peux maintenant voir les appels avec "SPAM : ..." dans ton carnet de contacts et à la sonnerie. Tu peux ainsi voir immédiatement si cela vaut la peine de répondre ou de rappeler.

Tu n'as plus tes données d'accès sous la main? Pas de problème, il suffit de réinitialiser le mot de passe dans les paramètres.

Tu trouveras également ailleurs un guide avec des images. Dans ces instructions, tu dois seulement utiliser les données d'accès PhoneBlock dans l'étape où le serveur, le nom d'utilisateur et le mot de passe sont saisis (voir ci-dessus).

Si tu n'utilises PhoneBlock que sur l'iPhone, tu peux aussi augmenter le nombre d'entrées dans les paramètres. Si cela pose des problèmes, merci de nous en faire part (voir FAQ).
