Εγκατάσταση Android

Η εγκατάσταση προσθέτει το βιβλίο διευθύνσεων του PhoneBlock στις επαφές σας. Εάν λάβετε μια κλήση από έναν αριθμό spam, αυτό θα εμφανίζεται αμέσως, π.χ. "SPAM: 03016637169". Μπορείτε επίσης να δείτε αμέσως στις αναπάντητες κλήσεις ότι δεν αξίζει να καλέσετε ξανά, χωρίς να χρειαστεί να γκουγκλάρετε πρώτα τον αριθμό.

Δεν μπορείτε να το χρησιμοποιήσετε για να μπλοκάρετε κλήσεις SPAM (το τηλέφωνό σας θα εξακολουθεί να χτυπάει, αλλά θα βλέπετε ότι πρόκειται για κλήση SPAM). Μπορείτε να μπλοκάρετε τις κλήσεις SPAM με την εγκατάσταση του SpamBlocker και την ενσωμάτωση της βάσης δεδομένων PhoneBlock. Το SpamBlocker είναι μια ανεξάρτητη εφαρμογή, παρακαλούμε επικοινωνήστε απευθείας με τον προγραμματιστή αν έχετε οποιοδήποτε πρόβλημα με αυτήν.

Για το συγχρονισμό του καταλόγου διευθύνσεων, χρειάζεστε μια εφαρμογή με την οποία μπορείτε να συγχρονίσετε τους τηλεφωνικούς καταλόγους του Διαδικτύου. Μπορείτε να χρησιμοποιήσετε το "PeopleSync", για παράδειγμα, ή το "DAVx⁵". Οι οδηγίες εδώ είναι γραμμένες για το "PeopleSync", αλλά η διαδικασία με το "DAVx⁵" είναι πολύ παρόμοια:
