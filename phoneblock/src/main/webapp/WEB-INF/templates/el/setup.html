Εγκατάσταση ως τηλεφωνικός κατάλογος

Για να εγκαταστήσετε τη λίστα αποκλεισμού, χρειάζεστε έναν δρομολογητή Internet "FRITZ!Box" της AVM και έναν λογαριασμό PhoneBlock. Δεν είναι απαραίτητο να είναι το πιο πρόσφατο μοντέλο, αλλά θα πρέπει να ελέγξετε αν είναι εγκατεστημένο σε αυτό το πιο πρόσφατο FRITZ!OS (07.29 ή νεότερο). Αν δεν ισχύει αυτό, χρησιμοποιήστε τις παρακάτω οδηγίες εγκατάστασης για να ελέγξετε αν η έκδοσή σας προσφέρει ήδη τις απαραίτητες επιλογές.

Σημείωση: Λόγω τεχνικών περιορισμών του Fritz!Box, δεν είναι δυνατή η φόρτωση ολόκληρου του καταλόγου μπλοκ σε έναν τηλεφωνικό κατάλογο. Επομένως, η ρύθμιση ενός Αποκριτή PhoneBlock προσφέρει καλύτερη προστασία από τις κλήσεις SPAM.

Τα στοιχεία σύνδεσής σας

Θα χρειαστείτε τα ακόλουθα δεδομένα για την εγκατάσταση.

Συνδεθείτε στο FRITZ!Box σας

Για να λειτουργήσει αυτό, πρέπει να βρίσκεστε στο σπίτι σας στο WLAN ή ο υπολογιστής σας πρέπει να είναι απευθείας συνδεδεμένος στο FRITZ!Box μέσω καλωδίου. Το FRITZ!Box θα σας ζητήσει πρώτα τον κωδικό πρόσβασής σας. Εάν δεν έχετε συνδεθεί ποτέ στο παρελθόν, θα βρείτε αυτόν τον κωδικό πρόσβασης στο κάτω μέρος της συσκευής.

Δημιουργία νέου τηλεφωνικού καταλόγου "Blocklist"

Για να μπορεί το FRITZ!Box σας να μπλοκάρει τις ανεπιθύμητες κλήσεις, χρειάζεται τον δικό του τηλεφωνικό κατάλογο στον οποίο καταχωρούνται όλοι οι αριθμοί των ανεπιθύμητων καλούντων. Καλέστε αυτόν τον τηλεφωνικό κατάλογο "Blocklist". Για αυτό

  1. Πλοηγηθείτε πρώτα στην ενότητα "Τηλεφωνία",
  2. επιλέξτε το στοιχείο μενού "Τηλεφωνικός κατάλογος" και
  3. και στη συνέχεια κάντε κλικ στο σύνδεσμο "Νέος τηλεφωνικός κατάλογος".

Εισάγετε τα στοιχεία σύνδεσης του λογαριασμού σας PhoneBlock

Τώρα χρειάζεστε τα στοιχεία σύνδεσης του λογαριασμού σας PhoneBlock.

  • 1 Δώστε στον τηλεφωνικό κατάλογο που πρόκειται να χρησιμοποιηθεί ως λίστα φραγής το όνομα Λίστα φραγής. Θα χρειαστείτε ξανά αυτό το όνομα για το επόμενο βήμα.
  • Τώρα επιλέξτε 2 την επιλογή Χρήση του τηλεφωνικού καταλόγου ενός διαδικτυακού παρόχου.
  • Στη συνέχεια, μπορείτε να επιλέξετε 3 τον πάροχο CardDAV provider.
  • Σχεδόν τελειώσαμε, τώρα τα δεδομένα σύνδεσης 4. Είναι καλύτερο να τα μεταφέρετε με Cut&Paste για να αποφύγετε τα λάθη πληκτρολόγησης!
    • Εισάγετε τη διεύθυνση URL του βιβλίου διευθύνσεων του PhoneBlock στο πεδίο Διεύθυνση διαδικτύου του διακομιστή CardDAV:
    • Εισάγετε το όνομα χρήστη που λάβατε κατά την εγγραφή στο πεδίο Όνομα χρήστη.
    • Ο κωδικός πρόσβασης που λάβατε κατά την εγγραφή, , πρέπει τώρα να εισαχθεί στο πεδίο Κωδικός πρόσβασης.
  • Τώρα μπορείτε να επιβεβαιώσετε 5 τη δημιουργία του νέου τηλεφωνικού καταλόγου. Παρακαλούμε σημειώστε: Επιλέξτε κανένα από τα τηλέφωνά σας στην ενότητα "Ανάθεση τηλεφώνου"! Διαφορετικά, θα βρίσκετε μόνο τους καλούντες SPAM στις επαφές του τηλεφώνου σας.

Ελέγξτε αν έχει δημιουργηθεί η "Blocklist" σας

Αν όλα πήγαν καλά, τότε το FRITZ!Box σας έχει φορτώσει όλους τους αριθμούς από τη λίστα αποκλεισμού PhoneBlock. Θα πρέπει τώρα να έχετε έναν νέο τηλεφωνικό κατάλογο "Blocklist" στην ενότητα "Telephony / Phonebook". Εάν επιλέξετε την καρτέλα "Blocklist", μπορείτε να δείτε όλους τους αριθμούς των τρεχόντων καλούντων spam.

Αφήστε το FRITZ!Box να απορρίψει κλήσεις από τη "Λίστα αποκλεισμού"

Στην ενότητα "Τηλεφωνία / Χειρισμός κλήσεων", μπορείτε τώρα να καθορίσετε τον νέο τηλεφωνικό κατάλογο "Blocklist" ως το εύρος αριθμών τηλεφώνου που θα μπλοκαριστεί.

Μετακινηθείτε προς τα κάτω στο υποστοιχείο "Block number ranges" και κάντε κλικ στο κουμπί "Add range".

Στη νέα οθόνη, επιλέξτε την επιλογή "Τηλεφωνικός κατάλογος" στην επιλογή για την "Περιοχή" και επιλέξτε τον πρόσφατα δημιουργημένο τηλεφωνικό κατάλογο "Blocklist" ως τον "Τηλεφωνικό κατάλογο" προς αποκλεισμό.
