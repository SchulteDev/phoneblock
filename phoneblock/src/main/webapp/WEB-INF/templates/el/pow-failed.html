Διακοπή σύνδεσης

Το PhoneBlock δεν μπορεί να επαληθεύσει ότι δεν πρόκειται για αυτοματοποιημένο μαζικό αίτημα. Ως εκ τούτου, το περιεχόμενο της σελίδας δεν μπορεί να εμφανιστεί. Εάν δεν είστε ρομπότ, μπορείτε να παρακάμψετε τον έλεγχο συνδεόμενοι στο PhoneBlock. Είναι προτιμότερο να ενεργοποιήσετε την επιλογή "παραμένετε συνδεδεμένοι" κατά τη σύνδεσή σας. Τότε αυτό το πρόβλημα δεν θα εμφανιστεί την επόμενη φορά που θα επισκεφθείτε τον ιστότοπο.

Ζητούμε συγγνώμη για την ταλαιπωρία, αλλά ο ιστότοπος PhoneBlock δέχεται επίθεση από έναν απίστευτο αριθμό υπολογιστών στο Διαδίκτυο με μαζικά ερωτήματα, τα οποία θα περιόριζαν σημαντικά τη διαθεσιμότητα του ιστότοπου χωρίς αυτόν τον έλεγχο. Για να μπορέσει το PhoneBlock να συνεχίσει να προσφέρεται δωρεάν (ή με δωρεά), αυτά τα μέτρα είναι δυστυχώς αναπόφευκτα.
