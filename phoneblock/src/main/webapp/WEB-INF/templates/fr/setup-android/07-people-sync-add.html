Installation d'Android

Étape 4 : Ajoute un carnet d'adresses et indique les données du compte PhoneBlock

  1. Choisis l'option "Se connecter avec l'URL et le nom d'utilisateur" et indique comme "URL de base" l'adresse du carnet d'adresses PhoneBlock : .
  2. Dans "Nom d'utilisateur", tu inscris le nom d'utilisateur que tu as reçu lors de l'inscription PhoneBlock ().
  3. Le mot de passe t'a été indiqué après la première connexion ().

Tu n'as plus tes données de connexion sous la main? Pas de problème, il suffit de se connecter et de réinitialiser le mot de passe dans les paramètres.
