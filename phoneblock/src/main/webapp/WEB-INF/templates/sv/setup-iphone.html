Installation av iPhone

Installationen lägger till PhoneBlocks adressbok i dina kontakter. Om du får ett samtal från ett skräppostnummer visas detta omedelbart, t.ex. "SPAM: 03016637169". Du kan också se direkt i de missade samtalen att det inte är värt att ringa tillbaka utan att behöva googla numret först.

Installationen sker i följande steg:

  1. Registrera dig med PhoneBlock.
  2. Öppna "Inställningar" > "Kontakter" > "Konton".
  3. Tryck på "Lägg till konto" - "Övrigt" > "Lägg till CardDAV-konto".
  4. Ange dina åtkomstuppgifter och tryck på "Nästa".
    Server
    https://phoneblock.net/kontakter/
    Användarnamn
    Logga in för att se ditt användarnamn.
    Lösenord
    Visas för dig direkt efter första inloggningen, du kan genera ett nytt i inställningarna
    Beskrivning
    PhoneBlock
  5. Aktivera "Kontakter" och tryck på "Spara".
  6. Nu ser du samtal med "SPAM: ..." i din kontaktbok och när de ringer. Du kan då direkt se om det är värt att svara eller ringa tillbaka.

Har du inte längre dina inloggningsuppgifter till hands? Det gör inget, du behöver bara återställa lösenordet i inställningarna.

Du kan också hitta instruktioner med bilder på andra ställen. I den här guiden behöver du bara använda PhoneBlock-åtkomstdata i det steg där du anger server, användarnamn och lösenord (se ovan).

Om du bara använder PhoneBlock på iPhone kan du också öka antalet poster i inställningarna. Om det uppstår några problem med detta, vänligen meddela oss (se FAQ).
